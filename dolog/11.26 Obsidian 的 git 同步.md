 Obsidian 有各种各样的同步策略：
 - 使用 iCloud —— 易冲突，不能看历史版本
 - 使用 Obsidian Sync —— 稳定可靠、堪比 Bear，但还是不能看历史版本
 - 使用 git 插件 —— 稍有门槛，但实属最佳方案

门槛在于——Git 插件踩坑：
- 一开始选的是 https://github.com/silvanocerza/github-gitless-sync ，配置坑太多，最终代码报错放弃
- 后面选 https://github.com/joshuakto/fit ，一切顺利。

经验：
- 插件让 AI 推荐可信度有限，卡壳就换，自己尝试也很有必要。

感触：
- 用 git 同步 Obsidian 实属最佳实践，很爽很爽。
