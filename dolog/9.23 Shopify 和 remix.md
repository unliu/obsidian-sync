# Dolog
#### 中国开发者的坑
感谢大模型，帮我快速踩完了下面的坑——cloudflare 不能正常访问，需要用 ngrok 绕过：
1. 注册并运行 ngrok： ngrok http 8000
2. 运行 shopify app dev --tunnel-url https://d58a485f4ea2.ngrok-free.app:8000
	1. 这里有个坑点是，它必须带端口参数
	2. 而要跑起来，需要同步修改 shopify.app.toml —— perplexity 没有告诉我这一点，还是 warp 偶然建议的。
