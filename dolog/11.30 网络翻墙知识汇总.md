## 翻墙最佳线路
早上好奇了解了一下自建翻墙的逻辑，了解到翻墙链路的最佳实践：
- 翻墙不顺利（丢包、不稳定）的原因有三：
	- 梯子质量有限：如日本使用 NTT 线路，其它地区没有使用 CN2 线路（CN2 和日本软银线路较为优选）
	- 国内运营商和梯子网络不匹配
		- 或者使用 CN2 线路的翻墙服务器，昂贵但很快
		- 或者使用中国联通 + 日本软银线路（不能是 NTT 线路）
	- AI 相关服务：IP 不干净导致封禁

## 补充了一下翻墙知识：

#### 基础概念：
- 翻墙的质量包括响应（ping）、丢包/抖动（Jitter）、上传下载速度三个方面

#### 纯净度和速度测试：
- 测速网站：
	- 老牌： https://www.speedtest.net/
	- 更全面： https://speed.cloudflare.com/
- 查运营商： https://bgp.he.net/
- 纯净度监测网站：
	- https://ping0.cc/ # 综合监测
	- https://scamalytics.com/ip # Fraud Check
	- https://www.ipqualityscore.com/ # Reputation & Risk
	- 针对性检测：OpenAI/Google 解锁检测：虽然 Ping0.cc 会简单标示是否解锁，但以下方式更准确：
		- OpenAI-Checker (脚本类)​
			- 如果你是自己搭建节点（VPS），可以用 GitHub 上的 missuo/OpenAI-Checker 脚本在服务器端直接跑测试，它能精确告诉你这个 IP 是否被 ChatGPT 屏蔽，以及支持的地区。
	- 直接访问测试 ChatGPT：访问 https://chatgpt.com/cdn-cgi/trace ，看 loc= 字段是否为你代理的国家代码，且页面无 Access Denied。
	- Google：在无登录状态下访问 google.com/ncr ，看底部定位是否准确显示为代理所在地，如果频繁跳出“我不是机器人”验证码，说明 IP 脏了。

**💡 注册成功的一条龙自测流程**
1. **挂代理**：开启全局模式（Global Mode）。
2. **查类型**：开 **IPinfo**，确认是 **ISP** 或 **Business**（尽量避开 Hosting）。
3. **查分**：开 **Scamalytics**，确保分数低于 30。
4. **查漏**：开 **Whoer**，调整时区和语言，争取凑到 100%。
5. **去注册**：使用指纹纯净的浏览器环境打开 ChatGPT/Google 注册页。
